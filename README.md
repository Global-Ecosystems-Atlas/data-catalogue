# data-catalogue
This repository houses all the relevant information about datasets assessed for, and synthesised into, the Global Ecosystems Atlas.

### In here you will find the following resources:

* Atlas [database]() - a tabular summary of all datasets assessed for inclusion in the Atlas, along with attributes and other metadata.
* Atlas [catalogue]() - tabular summary of all source datasets included in the Atlas.
* Separate [metadata]() files for each of dataset included in the Atlas (.json and .md formats are provided) 

### Additional resources
* For information about how these datasets have been cross referenced to the GET (including template and completed membership matrices), please see our [cross-referencing]() repository.
* For more information about the data synthesis process behind the Atlas, please see our [data-synthesis]() repository - *COMING SOON!*
